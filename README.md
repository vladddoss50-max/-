# ВСЁ ДЛЯ ФИПСИ от ВТ

Веб-приложение для статистических расчетов с поддержкой основных статистических критериев.

## Доступные критерии

1. **Критерий Фишера** - сравнение дисперсий двух выборок
2. **Критерий Стьюдента** - сравнение средних двух выборок
3. **Критерий Сиджела-Тьюки** - сравнение дисперсий (непараметрический)
4. **Критерий Манна-Уитни** - сравнение двух независимых выборок (непараметрический)
5. **Критерий Спирмена** - корреляционный анализ
6. **Критерий Хортлея** - обнаружение выбросов в данных
7. **Критерий Кочрена** - сравнение нескольких долей
8. **Критерий Бартлета** - проверка равенства дисперсий
9. **Однофакторный дисперсионный анализ** - сравнение средних нескольких выборок
10. **Корреляция Пирсона** - оценка линейной связи между переменными
